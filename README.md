# jsRemoveNodes

Research: Project4Dimensions

## Why jsRemoveNodes?

jsRemoveNodes are JavaScipt algorithms to remove HTML nodes that can be 
used to delete existing content and insert new material.

See [`js-remove-nodes.js`](js-remove-nodes.js).
